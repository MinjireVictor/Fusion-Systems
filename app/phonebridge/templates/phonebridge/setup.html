üìã Setup Instructions for New Server

Step 1: Update Zoho Developer Console

  1. Go to Zoho API Console
  2. Find your PhoneBridge application
  3. Update the Authorized Redirect URIs to:
    4. http://zoho.fusionsystems.co.ke:8000/phonebridge/zoho/callback/
  4. Save the changes in Zoho Console

Step 2: Verify Environment Configuration

  1. Check your .env file has the correct ZOHO_REDIRECT_URI
  2. Restart your Django application: python manage.py runserver 0.0.0.0:8000
  3. Test configuration using the buttons below

Step 3: Test OAuth Flow

  1. Use "Test OAuth" button below to verify configuration
  2. Try connecting to Zoho from the dashboard
  3. Check diagnostics if there are any issues
